### コミット時に自動で生計
```
yarn add -D husky lint-staged prettier
```
```package.json
 "scripts": {
    "precommit": "lint-staged"
  },
  "lint-staged": {
    "*.js": [
      "prettier-eslint --write",
      "eslint", // これを入れないとエラーがあってもコミットされてしまう
      "git add"
    ]
  },
```
